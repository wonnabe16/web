| ------------- | :----------------------------------------------: |
| `screen`      |         브라우저 외부 screen의 전체 크기         |
| `outer`       |           브라우저를 띄우는 가장 외곽            |

| `clientWidth` | 브라우저 내부의 contents영역(스크롤 ~~마포함~~)) |
